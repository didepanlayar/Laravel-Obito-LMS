## Description
**Obito Learning Management System** is a modern LMS built with Laravel 11, featuring seamless payment integration via Midtrans, an intuitive admin panel powered by Filament, and a clean architecture using the Service Repository pattern for scalability and maintainability.

## Tech Stack
- Laravel
- Git
- Docker
- Docker Compose
- Visual Studio Code
